
# Ideas

## Product ideas
- Texas Hold'em dealer assistant
- automated table-state reader
- mobile camera as input source
- poker hand / odds support
- chip counting
- demo-ready portfolio project

## Technical ideas
- contour-based card detection with OpenCV
- template matching for early rank/suit recognition
- PyTorch classifier for more robust recognition
- perspective correction for card normalization
- frame voting for stable live predictions

## Questions to explore
- Is template matching enough for early MVP?
- Should rank/suit recognition be split into separate steps?
- How should board cards and player cards be grouped?
- What image conditions are needed for reliable recognition?
- What is the best path from static image prototype to mobile demo?

## Game flow ideas
- turn timer for each player
- automatic blind increase after a defined time interval
- configurable table rules and round settings
